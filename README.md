# 🎮 2048 Game

A modern, feature-rich clone of the classic **2048** puzzle game built with **React**, **TypeScript**, and **Vite**. Combine matching tiles, beat your high score, and challenge yourself in multiple game modes with a polished, responsive interface.

---

##  Features

-  Classic 2048 gameplay
-  Timed Challenge Mode
-  Score & Best Score tracking
-  Automatic local save using Local Storage
-  Fully responsive design
-  Clean modern UI with smooth animations
-  Restart game anytime
-  Keyboard controls
-  Swipe controls for mobile devices
-  Sound effects with mute option
-  Win and Game Over overlays
-  Fast performance powered by React + Vite

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/prxns/2048-game.git
```

Navigate to the project

```bash
cd 2048-game
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# 🧠 Gameplay Rules

- Every move spawns a new tile.
- Merge identical tiles to create higher values.
- Reach **2048** to win.
- The game ends when no valid moves remain.
